# 추가할 기능

- 키워드 설정이나 그런 것을 설정하면 알림 받을 수 있게

- 검색 기능

- 채팅기능으로 강아지나 고양이를 올린 사람한테 채팅을 걸 수 있는 

- 찾으면 인증을 통해 기록으로 변경되고 현재 실시간인 내용만 뜨는 느낌

- 오른쪽에 신고 목록이 뜨고 

- 찾는 사람도 게시글을 올려서 찾을 수 있게

- 그리고 찾는 게시글이 비슷한게 올라오면 매칭을 통해 알림이나 그런 것을 할 수 있게.

- 그리고 비슷한 글이(판단을 통해 이거 저 게시글의 동물이랑 비슷한데? 그러면 사용자에게 물어봐서 혹시 이거랑 같은 동물이니 확인 후 진행) 올라오면 카테고리처럼 묶어서 보여줄 수 있게 하고

- 카테고리로 묶은 게시글을 통해 위치가 어떻게 이동하고 있는지 보여줄 수 있게.
